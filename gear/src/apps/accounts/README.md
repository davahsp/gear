# Accounts Module
### Overview

This module is responsible for:

* User authentication

* Role-Based Access Control (RBAC)

* User and role management

* Password reset

This module serves as the foundation of Sprint 1 and becomes a dependency for features implemented in Sprint 2 and Sprint 3.

### System Roles (As Defined in the PRD)

The system defines the following roles:

* `Owner`

* `Sales`

* `FinanceAdmin`

* `Warehouse`

* `Customer`

Each role has restricted access to operational modules, financial modules, and analytical dashboards according to business requirements.

### Sprint 1 Scope

* Login & logout

* Role-Based Access Control (RBAC)

* Password reset

* Feature access restriction based on role
